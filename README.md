# Lawn Mower

Nous souhaitons développer une tondeuse à gazon automatique, destinée aux surfaces rectangulaires.
La tondeuse peut être programmée pour parcourir l'intégralité de la surface.

# Version
- Spring Boot 2.6.6
- Java 11
- Maven 3.5.1
